# RAG-based ML Q&A Chatbot

This project implements a Retrieval-Augmented Generation (RAG) chatbot using Ollama.

## Features
- Text chunking and embedding generation
- Cosine similarity-based retrieval
- LLM-based response generation (Gemma)
- Streamlit interface for real-time interaction

## Tech Stack
- Python
- Ollama
- Streamlit
- NumPy

## How to Run
1. Install dependencies
2. Run Streamlit app
3. Ask questions on custom dataset
